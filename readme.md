## 📐 Operações Básicas

**ex1.c** — Lê um número real e imprime a **quinta parte** (`n / 5`)

**ex2.c** — Lê **4 floats** e calcula a **média aritmética**

**ex3.c** — Lê **idade** e **ano atual**, calcula o **ano de nascimento**

**ex4.c** — Lê os catetos **a** e **b**, calcula a **hipotenusa** com `sqrt(a²+b²)`

## 🔀 Condicional — `if / else / switch`

**ex5.c** — Lê 2 inteiros → imprime o **maior** (ou "Numeros iguais")

**ex6.c** — Lê salário e prestação → informa se **empréstimo é concedido** (limite: 20% do salário)

**ex7.c** — Lê idade de nadador → classifica em categoria:
- 5–7 → Infantil A
- 8–10 → Infantil B
- 11–13 → Juvenil A
- 14–17 → Juvenil B
- 18+ → Sênior

**ex8.c** — Lê número de 1 a 7 → imprime o **dia da semana** (`switch`)

---

## 🔁 Repetição — `for / while`

**ex9.c** — Lê N → imprime naturais de **0 até N** (crescente)

**ex10.c** — Lê N → imprime naturais de **N até 0** (decrescente)

**ex11.c** — Lê N → imprime os **N primeiros ímpares** (`2*i + 1`)

**ex12.c** — Calcula a **soma dos 50 primeiros pares** (sem entrada do usuário)

**ex13.c** — Lê **10 inteiros positivos** (ignora não-positivos) → imprime a **média** (`while`)

**ex14.c** — Lê número positivo → lista todos os seus **divisores**

**ex15.c** — Lê número → calcula a **soma dos divisores** (exceto ele mesmo)

**ex16.c** — Lê N → imprime o **N-ésimo termo de Fibonacci** (iterativo)

**ex17.c** — Lê N → imprime N linhas do **triângulo de Floyd** (dois `for` aninhados)

**ex18.c** — Lê N → calcula **E = 1/1! + 1/2! + ... + 1/N!** (dois `for` aninhados)

**ex19.c** — Loop infinito que repete até o usuário digitar **123** (`break`)

**ex20.c** — Função `maior(a, b)` → retorna o **maior entre dois inteiros**

**ex21.c** — Função `IMC(peso, altura)` → calcula e retorna o **IMC** (`peso / altura²`)

**ex22.c** — Função `fnum(n1, n2, op)` → executa **+, -, *, /** conforme o símbolo recebido

**ex23.c** — Função **recursiva** `somatorio(N)` → retorna a soma de 1 até N
